> Jest configs

### Install

```bash
yarn add -D @boilerz/jest-config 
```

### Usage

`jest.config.js` example:

```js
module.exports = require('@boilerz/jest-config');
```

Additional configs available:
- `@boilerz/jest-config/frontend`
- `@boilerz/jest-config/react-native`
